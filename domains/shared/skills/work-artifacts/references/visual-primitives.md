# Visual Primitives

Artifacts use markdown-native primitives so future renderers can parse them consistently.

## Approved

- Mermaid for sequence, flow, state, and architecture diagrams.
- tables for matrices, comparison grids, and Quality Contract ladders.
- structured lists for decision logs, risks, assumptions, and boundary rules.
- checklists for acceptance criteria and reviewable completion criteria.

## Forbidden

ASCII-art diagrams are forbidden. Do not draw boxes, arrows, or dependency maps with text glyphs. Use Mermaid or a table instead.

## Selection

- Use Mermaid when relationships or flow matter.
- Use tables when comparing fields across rows.
- Use structured lists when order and nested rationale matter.
- Use checklists when the reader needs completion state.
